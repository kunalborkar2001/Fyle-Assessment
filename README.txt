Tax Calculator Web Application

Overview

This web application is designed to calculate taxes based on user input. Below, you'll find details on the application's functionality, features, and how to get started.

Application Features

Allows users to input gross annual income, extra income, age group, and deductions.
Calculates taxes based on the provided inputs and Indian tax laws.
Implements error handling for incorrect input values, displaying error icons and tooltips for guidance.
Displays the final calculated tax in a modal upon submission.



Getting Started

To get started with the Tax Calculator web application, follow these steps:
Clone the repository to your local machine:
bash
Copy code
git clone <repository-url>
Open the index.html file in your preferred web browser.

Input your gross annual income, extra income, age group, and deductions into the respective fields.

Ensure to select the appropriate age group from the dropdown menu.

Click on the "Calculate Tax" button to see the calculated tax.

If there are any errors in the input fields, error icons will appear with tooltips for guidance.

Assumptions

Deductions include allowable tax deductions such as investments, donations, etc.
Age is calculated as of the current year.
The tax calculation follows Indian tax laws.
